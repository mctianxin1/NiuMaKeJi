# 牛马科技
- 这是一个基于rsc插件写的slimefun附属
- 增加许多自动化生成的机器
- 同时也非常的肝
- 这个附属不会像海曼科技的分子重组仪那样制造一些材料就随随便便让玩家等待几个小时
- 附属的前置有:
  - Slimefun
  - InfinityExpansion
  - FinalTECH
  - RykenSlimeCustomizer
  - FoxyMachines
